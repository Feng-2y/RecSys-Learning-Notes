# RecSys-Summer-Learning
> 大一升大二暑假推荐系统系统学习仓库，基于Python + PyTorch实现，包含传统推荐算法、深度推荐模型与完整工程项目。
## 📖 项目简介
本人已有线性代数、概率论、数据结构与算法基础，利用30天暑假系统学习推荐系统(RecSys)。
学习路线：传统协同过滤 → 隐语义模型 → FM深度推荐模型 → 多路召回+精排完整推荐系统。
同时实践Vibe Coding开发模式，借助AI工具加速工程实现，重心放在算法原理推导。

## 🗂️ 目录结构

├── README.md

## ✅ 学习目标
1. 掌握协同过滤、矩阵分解、FM等经典推荐算法，能够公式推导+手写代码实现
2. 理解工业界「召回-排序」两级架构，掌握多路召回策略
3. 熟练使用PyTorch搭建主流深度CTR预估模型
4. 完成一套可演示的端到端个性化推荐系统，作为简历/大创项目
